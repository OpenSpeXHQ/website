# OpenSpeX Website

This ZIP contains a static OpenSpeX landing website built with HTML, CSS, vanilla JavaScript, and locally generated PNG images.

## Files

- `index.html` — single-page website
- `styles.css` — full responsive styling
- `script.js` — smooth scrolling, active navigation, reveal effects, and small counters
- `assets/` — generated PNG visuals and favicon

## How to open locally

1. Unzip `OpenSpeX_Website.zip`.
2. Open `index.html` in a modern browser.
3. No install step is required.

The website works offline after extraction.
